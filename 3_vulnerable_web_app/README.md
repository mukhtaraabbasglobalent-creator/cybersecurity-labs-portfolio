# Vulnerable Web Application Security Lab

## Objective
To simulate a real-world web application security assessment
using a deliberately vulnerable application in a controlled environment.

## Scope
- Test common web vulnerabilities
- Document findings professionally
- Demonstrate ethical hacking methodology

## Tools Used
- Kali Linux
- DVWA (Damn Vulnerable Web Application)
- Browser
- Linux Terminal

## Environment
- Attacker: Kali Linux VM
- Target: DVWA (local / isolated)
- Network: Host-only / Lab network

## Ethical Statement
All testing was performed on systems created for learning purposes.
No real-world systems were targeted.
