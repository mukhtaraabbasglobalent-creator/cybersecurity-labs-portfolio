# Kali Linux Virtual Machine Lab Report

## Objective
To set up a Kali Linux virtual machine for safe and legal penetration testing practice.

## Environment
- Host OS: Windows / Linux (edit this)
- Virtualization: VirtualBox
- Guest OS: Kali Linux
- RAM: 4GB
- Storage: 50GB

## Tools Used
- Kali Linux
- Linux Terminal
- Nmap
- Metasploit Framework

## Procedure
1. Downloaded Kali Linux ISO from the official website
2. Created a new virtual machine in VirtualBox
3. Installed Kali Linux successfully
4. Updated system packages using terminal
5. Verified tools by running basic commands
6. Performed a sample Nmap scan in an isolated environment

## Results
- Kali Linux installed successfully
- System updated without errors
- Nmap scan executed correctly
- Metasploit framework initialized
